# GridMe デザインシステム

Z世代向けのモダンで「シンプルだけどワクワクする」デザインシステムへようこそ！

## 🎨 デザインフィロソフィー

GridMeは、以下の4つの原則に基づいて設計されています：

### 1. **Authenticity（真正性）**
誇張しない、素直なデザイン。ユーザーの本音と向き合い、透明性のあるインターフェースを提供します。

### 2. **Fluidity（流動性）**
スムーズなトランジションと自然な動きのマイクロインタラクション。レスポンシブで適応的なレイアウトを実現します。

### 3. **Playfulness（遊び心）**
予想外の楽しい要素とインタラクティブなフィードバック。エモーショナルなつながりを生み出します。

### 4. **Minimalism Plus（ミニマリズム・プラス）**
必要最小限の要素に、意味のあるアクセントをプラス。ネガティブスペースを活用したクリーンなデザイン。

## 🚀 使い方

### 1. デザイントークンの読み込み
```html
<link rel="stylesheet" href="/styles/design-tokens.css">
```

### 2. コンポーネントスタイルの読み込み
```html
<link rel="stylesheet" href="/styles/components.css">
```

### 3. コンポーネントの使用
```html
<button class="btn-primary">はじめる</button>
```

## 📚 コンポーネント

左側のサイドバーから、各コンポーネントの詳細とバリエーションをご覧いただけます。

- **Button**: 様々なスタイルのボタンコンポーネント
- **Card**: 情報を整理して表示するカードコンポーネント
- **Form**: 美しく機能的なフォーム要素
